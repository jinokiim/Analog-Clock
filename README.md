# 아날로그 시계

React, Redux 및 Vite를 사용하여 구현된 아날로그 시계 프로젝트입니다.
현재 시간을 기준 시침, 분침, 초침을 확인 할 수 있습니다.
시계에 마우스 hover시 현재 시간을 툴팁으로 확인할 수 있으며 마우스를 따라다닙니다.

- Node.js v21.4.0
- npm -v10.2.4
- pnpm : 8.14.1

# 개발 환경

- React: 18.2.0
- typescript: 4.9.5
- redux
- redux toolkit


# 저장소 Clone

```bash
$ git clone https://github.com/jinokiim/Analog-Clock.git
$ cd Analog-Clock
```

# 설치

```
$ pnpm install
```

# 로컬 서버 시작

```
$ pnpm start
```
